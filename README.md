# landtagszusammenfasser_bawue_scraper

Baden-Württembergisches Parlement Scraper für den Landtagszusammenfasser.
Der Landtagszusammenfasser möchte die Gesetze in den unterschiedlichen Landtagen und unterschiedlichen Stadien transparent machen.
Dazu benötigt er verschiedene scraper für jeden deutschen Landtag und ggf. auch die Ministerien.

Der Bawue-Scraper ist einer dieser Scraper mit dem Fokus auf den Baden-Württembergischen Landtag.
